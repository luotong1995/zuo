### 删除链表中间节点

1. 只有一个节点则不删除节点
2. 有两个节点就直接删除第一个节点
    1->2   : 删除1
3. 1->2->3 :删除2
   1->2->3->4 :删除2
   1->2->3->4->5 :删除3
   1->2->3->4->5->6 :删除3
    
   则可以发现链表长度每增加2需要删除的节点就向后移一位


### 输入a b 删除位于a/b 的节点
n为链表的长度 
r = a/b

r = 0 不删除

0<r<=1/n 删除1节点

1/n<r<=2/n 删除2节点

2/n<r<=3/n 删除3节点

变换一下即

0< n*r <=1 删除1节点

1< n*r <=2 删除2节点

2< n*r <=3 删除3节点

n*r向上取整即为需要删除的节点
